**RbnDeploy** is a powerful, all-in-one utility designed to setup, optimize, and maintain your Windows installation. Whether you've just formatted your PC or want to clean up an existing system, RbnDeploy automates the tedious tasks for you.

## ‚ú® Key Features

### üõ†Ô∏è System Optimization (Deploy)
- **Debloat Windows**: Remove pre-installed junk like OneDrive, Teams, Cortana, and more.
- **Smart Detection**: Automatically detects installed bloatware and only offers relevant cleanup options.
- **Telemetry & Privacy**: Disable Windows spying/telemetry services with one click.
- **Visual Feedback**: Real-time progress updates and "Success" verification.

### üéÆ Drivers & Update
- **GPU Drivers**: Checks for the latest NVIDIA, AMD, and Intel drivers.
- **BIOS Updates**: Detects your motherboard model and directs you to the exact update page.
- **Windows Activation**: Checks license status securely.

### üì¶ Software Manager
- **One-Click Install**: Install essential apps (Chrome, Discord, Steam, VS Code, etc.) using `winget`.
- **Bulk Installation**: Select multiple apps and install them all at once.

### ‚ö° Performance Monitor
- **Real-Time Stats**: Monitor CPU Temp, RAM Usage, and Disk Activity.
- **Deep Analysis**: Detailed hardware reporting (Core counts, Threads, Manufacturer info).
